# Resize event

Create markup like this:

```html
<div class="task-wrapper">
    <p>Resize window</p>
    <div class="out">0</div>
</div>
```

Write a function which increases initial value in `.out` every time when you change `size(width, height)` of browser window. Use `toggle device toolbar`
