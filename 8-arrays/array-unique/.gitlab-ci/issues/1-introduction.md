# Array Unique

Implement the function `unique` that accepts an `array`.
The function should return new `array` with unique items of given `array`.

## Test Examples

```javascript
expect(run([1, 1, 2, 3, 2])).toBe([1, 2, 3]);
expect(run([5, 5, 5, 5, 5, 5])).toBe([5]);
```
