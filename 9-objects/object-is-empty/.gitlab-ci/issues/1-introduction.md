# Is Empty Object

Write the function `isEmpty(obj)` which returns `true` if the object has no properties, `false` otherwise. If an argument is not an `object`, return `support only objects`

## Test Examples

```javascript
expect(isEmpty({})).toBe(true);
expect(isEmpty({ name: 'David' })).toBe(false);
```
