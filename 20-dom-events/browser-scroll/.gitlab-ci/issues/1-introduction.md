# Scroll event

Create markup like this:

```html
    <div class="task-wrapper">
        <p>Scroll event</p>
        <div class="out">1</div>
    </div>
```

Write a function that increases the initial value in `.out` whenever you make `scroll(scrollUp, scrollDown)`
